Java EE MicroProfile ve MikroServisler

Monolit uygulama yapılarından Mikroservis mimarisine geçişin anlatılacağı bu sunumda, Java EE MicroProfile uyumlu sunucular ile Mikroservis gerçekleştirimleri örneklendirilecek ve Kurumsal Java' da ortaya koyulan bu çabanın gidiş yönü yeni spesifikasyonlar ile detaylandırılacaktır.

Web Uygulamalarında Performans Ölçümü ve Optimizasyonu

Bu konuşmada modern web uygulamalarının performansına etki eden faktörler ve karşılaşılan problemlere ek olarak performans ölçümü ve optimizasyonu üzerinde durulacaktır. Günümüzde kullanılmakta olan tarayıcıların, performans ölçümü amacıyla sunduğu API ve bunların kullanımı yanı sıra server-side/client-side optimizasyon yöntemleri konuşmada yer alacaktır.

Anahtar Kelimeler

Front-end performance, critical rendering path, javascript bundling and minification, HTTP/2 server-push, User Timing API, Resource Timing API
